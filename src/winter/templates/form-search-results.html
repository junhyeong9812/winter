🔧 기술 정보:
• 컨트롤러: @Controller SearchController
• 매핑: @RequestMapping(value="/search/form", method="POST")
• 파라미터: @ModelAttribute SearchForm searchForm, HttpResponse response
• 특징: ModelAttributeBinder를 통한 자동 객체 바인딩
• 바인딩 과정:
  1. HTTP 요청 파라미터들을 Map으로 수집
  2. SearchForm 객체 생성 (기본 생성자)
  3. setter 메서드를 리플렉션으로 찾아 자동 호출
  4. 완성된 객체를 메서드 파라미터로 전달
• 유효성 검증: keyword 필수값 체크 후 400 에러 처리
