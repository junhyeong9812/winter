🔍 Advanced Search Results

고급 검색 완료!

여러 @RequestParam을 조합한 복합 검색 테스트

검색 키워드 ${keyword}
카테고리 ${category}
최소 가격 ${minPrice != null ? minPrice : '제한 없음'}
최대 가격 ${maxPrice != null ? maxPrice : '제한 없음'}
정렬 기준 ${sortBy}

🎯 검색 조건 요약

${searchSummary}

📊 검색 결과

${resultCount}개의 항목을 찾았습니다.

🔧 기술 정보:
• 컨트롤러: @Controller SearchController
• 매핑: @RequestMapping(value="/search/advanced", method="GET")
• 파라미터 특징:
  - @RequestParam("keyword") String keyword (필수)
  - @RequestParam(value="category", defaultValue="all") String category (기본값)
  - @RequestParam(value="minPrice", required=false) Integer minPrice (선택적)
  - @RequestParam(value="maxPrice", required=false) Integer maxPrice (선택적)
  - @RequestParam(value="sortBy", defaultValue="relevance") String sortBy (기본값)
• 특징: 필수/선택적 파라미터, 기본값, null 처리 조합 테스트
